This is a list of http status codes Code: taken almost all from here wikipedia . When you click on the number, a page with the code description will open up for you to see. Under the cover, the page will send the code you clicked to your browser. If you want to use the codes directly without having to use the interface, like when using postman, then just call the code directly like that: "http://fc572.herokuapp.com/httpcodes/200". Happy testing!
